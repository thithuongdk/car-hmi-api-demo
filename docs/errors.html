Error Codes Reference

All API errors return this JSON format:

{
  "error": "Short human-readable message",
  "code": 3002,
  "id": "VAL_MISSING_FIELD"
}

Optimistic Locking

For PUT /api/profile and PUT /config, client must send the latest section_id. If stale, server returns 409 VAL_CONFLICT.
